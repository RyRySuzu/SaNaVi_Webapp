# SaNaVi_Webapp
スマートフォンを用いて船舶事故を防止するシステムです。  
小型船舶の事故対策が現状できていない、という問題を解決するために開発しました。  
「船舶情報表示機能」「接近検知機能」「遭難検知機能」「ドライブレコーダー機能」の4つの機能で船舶事故を防止します。  
諸々の事情でPHPはコミットしてません。  
第三回学生スマートフォンアプリコンテスト　優勝  
ImagineCup2016 日本予選大会　ワールドシチズンシップ部門　部門賞
